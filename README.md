# Расписание Университета

Минималистичный сайт расписания занятий университета в стиле ChatGPT Pro.

## Технологии

- React 18
- TypeScript
- Vite
- React Router

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

## Структура

- **Экран 1**: Список факультетов
- **Экран 2**: Список групп выбранного факультета
- **Экран 3**: Расписание группы (числитель/знаменатель)
- **Экран 4**: Админ панель для редактирования расписания

## Дизайн

- Темная тема (черный/серый/белый)
- Круглые кнопки
- Минималистичный стиль
- Адаптивный дизайн (мобильные устройства 9:16 и десктоп)
- Футер с копирайтом на всех экранах
